# Драйвер принтера Panasonic KX-MB1500 для macOS (`x86_64` + `arm64`)

Это неофициальный драйвер печати для серии принтера `Panasonic KX-MB1500`.

Печать здесь сделана своим CUPS-фильтром на исходниках линукса, нейро-слопом.
Драйвер печати собран как universal пакет и работает как на `Intel` так и на `Apple Silicon`.

## Только печать

Если нужна только печать:

1. Подключи принтер по USB.
2. Скачай наш [.pkg из последнего релиза](https://github.com/MKC-MKC/driver-panasonic-kx-mb1500-macos/releases/latest).
3. Открой пакет.
4. Сработает защита macOS от непроверенных разработчиков.
5. Открой настройки Mac.
   ![](artifacts/img/1.jpg)
6. Конфиденциальность и безопасность.
7. Прокрути в самый низ, найди кнопку "Все равно открыть".
8. Вернись к установщику и пройди установку (macOS может запросить пароль). На шаге **«Настроить»** для режима «только печать» оставь включённым один пункт **Драйвер печати**, остальные сними.
9. Открой в настройках "Принтеры и сканеры" (в самом низу).
   ![](artifacts/img/2.jpg)
10. Нажми "Добавить принтер, сканер или факс..."
11. В списке появится принтер. Выбери его.
12. Снизу будет "Имя", "Размещение", "Использовать": Выбери в "Использовать:" "Выбрать ПО...".
13. Вверху будет поиск по фильтру. Напиши `Panasonic GDI` или `Haikiri`.
14. Выбери драйвер, далее "ОК", потом "Добавить".
15. Если принтер пишет "ПК печатает", нажми "STOP".
16. Перезагрузи принтер.
17. Готово.

## Печать и сканер

Для сканирования нужен старый официальный Panasonic backend app.
На `Apple Silicon` он работает только через `Rosetta`!

Официальные ссылки Panasonic:

- [Mac_1.15.2.dmg](https://www.psn-web.net/cs/support/fax/common/file/Mac_Installer/Mac_1.15.2.dmg)
- [Страница драйвера](https://docs.connect.panasonic.com/pcc/support/fax/common/table/macdriver.html)

Копию живого файла приложил в артефакты, если источник вдруг станет недоступен.

Готовый [.pkg из релиза](https://github.com/MKC-MKC/driver-panasonic-kx-mb1500-macos/releases/latest) уже включает установщик Panasonic с диска `Mac_1.15.2` и шаг, который подстраивает программу сканера под современные macOS, чтобы она нормально открывалась в «Захвате изображений». Отдельно качать DMG обычно не нужно.

> Было так: ставишь DMG Panasonic, перезагружаешься, потом **ещё раз** запускаешь наш pkg. Сейчас достаточно **одного** запуска нашего pkg с нужными галочками в «Настроить» (сканер + исправление, либо все три пункта для полной установки).

Алкоритм установки:

1. Подкинь принтер по USB.
2. Если Mac на `Apple Silicon`, установи `Rosetta` (`softwareupdate --install-rosetta --agree-to-license` в Терминале).
3. Скачай [.pkg из релиза](https://github.com/MKC-MKC/driver-panasonic-kx-mb1500-macos/releases/latest) и открой его. Если macOS **не даёт открыть** файл — сделай, как на скриншотах в разделе **«Только печать»**: **Настройки Mac** → **Конфиденциальность и безопасность** → внизу **«Всё равно открыть»** (`artifacts/img/1.jpg` и следующие картинки там же), затем снова запусти установщик.
4. Дальше в установщике пройди шаги до конца (macOS может запросить пароль). На экране **«Настроить»** отметь нужное — **по умолчанию всё включено**, это **полная установка**:
   - **Драйвер печати** — наш драйвер для печати через системный список принтеров.
   - **Panasonic Multi-Function Station** — официальный пакет сканера из `Mac_1.15.2`. После него Mac **может попросить перезагрузку** — так было и у Panasonic, это ожидаемо.
   - **Исправить ПО Panasonic** — донастройка установленной программы сканера, чтобы **«Захват изображений»** не падал на новых macOS. Пункт **активен**, если ты отмечаешь установку MFS **или** эта программа Panasonic уже стоит на компьютере.
5. Если принтер пишет «ПК печатает», нажми `STOP`, перезагрузи принтер.
6. Готово: открой **«Захват изображений»** и проверь сканер.
